Installation

Install from PyPI and start using LLM SuperCLI in a few commands.

1. Install the package

pip install llm-supercli

2. Authenticate (optional but recommended)

llm-supercli auth login --provider github
# or
llm-supercli auth login --provider google

3. Run your first command

llm-supercli run "Summarize the last 100 lines of logs in ./logs/app.log"

llm-supercli tools list
llm-supercli help
